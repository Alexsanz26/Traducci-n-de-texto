# Traducci-n-de-texto
Este proyecto es una clonación de la plataforma de videollamadas Zoom, desarrollada con tecnologías web modernas como HTML, CSS, JavaScript y Node.js. El objetivo principal es replicar algunas de las funcionalidades clave de Zoom, como videollamadas, chat integrado, compartir pantalla y colaboración en equipo, además de incluir un chatbot.
